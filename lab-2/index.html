CSS Positioning

Used to alter the element's natural layout flow position on the page and customize it according to our needs. It can be used to fix an eleemnt somewhere on the page, give a relative or an absolute position, or no special position at all.

Static

Static positioned elements are not affected by the top, bottom, left, and right porperties.

This is a static positioned div.

Fixed

This is a fixed div
This div element has position: relative;

Absolute

Absolute behaves like fixed except relative to the nearest positioned ancestor instead of relative to the viewport. If an absolutely-positioned element has no positioned ancestors, it uses the document body and still moves along with page scrolling.

Sticky

Sticky is a hybrid between relative and fixed position, which allows an element to act like it is relatively positioned until is is scrolled to a certain point.
